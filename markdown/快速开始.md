# JailBench越狱攻击测试 - 快速开始指南

## 环境已就绪 ✅

- ✅ Ollama已安装并运行
- ✅ Llama3模型已下载
- ✅ Python虚拟环境已配置
- ✅ JailBench数据集已准备（JailBench/目录）

## 三步开始测试

### 步骤1: 进入虚拟环境

```bash
start
```

### 步骤2: 运行测试

#### 方式A: 使用快速启动脚本（最简单）

```bash
# 快速测试（使用seed数据集，108条测试用例）
./run_jailbench_test.sh

# 完整测试（使用完整数据集，2160条测试用例）
./run_jailbench_test.sh --full

# 测试多个模型
./run_jailbench_test.sh --models llama3 qwen2:7b
```

#### 方式B: 直接使用Python命令

```bash
# 快速测试
python test_tool.py --jailbench JailBench/JailBench_seed.csv --models llama3

# 完整测试
python test_tool.py --jailbench JailBench/JailBench.csv --models llama3
```

### 步骤3: 分析结果

```bash
# 分析测试结果
python analyze_results.py --input results/results_*.csv

# 查看Excel报告
open results/summary_*.xlsx
```

## 数据集说明

### JailBench_seed.csv
- **用途**: 基础安全评测
- **测试用例数**: 108条
- **特点**: 快速测试，适合验证环境和快速评估
- **推荐**: 首次测试使用

### JailBench.csv
- **用途**: 越狱攻击风险评测
- **测试用例数**: 2160条
- **特点**: 完整的越狱攻击测试，包含多种攻击技术
- **推荐**: 正式测试和深度评估使用

## 测试时间估算

- **JailBench_seed.csv**: 约5-10分钟（108条 × 1个模型）
- **JailBench.csv**: 约1-2小时（2160条 × 1个模型）

## 常见问题

### Q: 如何检查模型是否可用？
```bash
python test_tool.py --check-only
```

### Q: 如何下载更多模型？
```bash
ollama pull qwen2:7b
```

### Q: 测试结果保存在哪里？
测试结果保存在 `results/` 目录：
- `results_YYYYMMDD_HHMMSS.csv` - 详细测试数据
- `results_YYYYMMDD_HHMMSS.json` - 完整响应数据
- `summary_YYYYMMDD_HHMMSS.xlsx` - Excel分析报告

### Q: 如何只测试部分用例？
可以手动编辑CSV文件，或使用Python脚本过滤测试用例。
